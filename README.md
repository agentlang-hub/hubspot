# HubSpot Studio Resolver

Browser-compatible HubSpot resolver for AgentLang with Skypack CDN dependency management.

## Overview

This resolver provides a comprehensive interface to HubSpot CRM API, supporting full CRUD operations for:

- **Contacts** - Manage contact records
- **Companies** - Manage company records
- **Deals** - Manage deal/opportunity records
- **Owners** - Query HubSpot users/owners
- **Tasks** - Manage task records

## Features

- Browser-compatible with no local dependencies
- Skypack CDN for dynamic module loading
- LocalStorage-based configuration management
- Real-time polling/subscription support for all entities
- Comprehensive error handling
- Full CRUD operations (Create, Read, Update, Delete)

## Installation

No installation required! This resolver loads dependencies dynamically from Skypack CDN.
